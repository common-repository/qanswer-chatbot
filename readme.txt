== Frequently Asked Questions ==
= How do I setup my chatbot? =

First you should have a QA system on [https://app.qanswer.ai](https://app.qanswer.ai/).
If you don't, you can create one. Do not forget to make it public, else you will not be able to put it on your website.
You can find the documentation at [https://doc.qanswer.ai](https://doc.qanswer.ai/).
Then you just have to fill in your username and the name of your QA system in the settings of the plugin, in the admin dashboard in `Settings > QAnswer`.
